## 📖 The Vision
Architecture is not merely about constructing buildings; it is about harmonizing human urgency with the integrity of the land. **ARKITEKT** began as an experiment: *How can we translate the weight of concrete, the flow of air, and the precision of minimalist design into the fluidity of web development?*

This repository serves as a digital blueprint. It explores the intersection of high-end architectural aesthetics and modern web interactivity, creating a space where the user doesn't just "browse"—they experience the architecture.

## ✨ Features

- **Kinetic Interaction**: Immersive custom cursor mechanics and 3D tilt-effects on project cards, adding physical tactility to the browser.
- **Scroll-Triggered Narrative**: Leveraging **GSAP** to reveal content dynamically, guiding the user through the studio’s portfolio with purpose.
- **Fluid Navigation**: Built with **Lenis** smooth scrolling for a seamless, "weightless" user experience.
- **Utility-First Aesthetics**: Designed with **Tailwind CSS** to maintain a rigorous, clean, and responsive grid system.

## 🏗️ Project Anatomy

The codebase is divided into three essential pillars that constitute the ARKITEKT digital environment:

| File | Role |
| :--- | :--- |
| `index.html` | **The Foundation.** The structural blueprint that dictates the semantic flow of the content. |
| `style.css` | **The Aesthetic Layer.** Custom overrides and subtle visual accents that define the "minimalist tropical" mood. |
| `script.js` | **The Kinetic Energy.** The logic behind the loaders, scroll animations, and interactive DOM behaviors. |
