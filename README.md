## 📖 Tentang

> **ARKITEKT** adalah proyek web kontemporer yang mengeksplorasi jembatan antara estetika arsitektur minimalis dengan interaksi web modern. Proyek ini dibangun sebagai wadah pembelajaran untuk menguasai pengembangan front-end yang kompleks, menggunakan **Tailwind CSS** untuk desain responsif dan pustaka animasi **GSAP** serta **Lenis** untuk menciptakan pengalaman pengguna yang mulus[cite: 9].

Proyek ini tidak hanya menampilkan portofolio studio, tetapi juga sebuah narasi visual yang presisi mengenai ruang, material, dan konteks arsitektural[cite: 9].

## 🛠️ Instalasi

Pastikan Anda memiliki lingkungan lokal untuk menjalankan file HTML.


# 1. Clone repositori
```bash
git clone [https://github.com/username/arkitekt.git](https://github.com/username/arkitekt.git)
```

# 2. Masuk ke direktori
```bash
cd arkitekt
```
# 3. Buka index_3.html di browser Anda
Pro-tip: Gunakan ekstensi "Live Server" di VS Code untuk pengalaman pengembangan yang lebih efisien.📁 Struktur ProyekStruktur direktori dirancang agar modular dan mudah dipahami:Plaintextarkitekt/
```
├── assets/
│   └── (images...)
├── css/
│   └── style.css               # Penyesuaian gaya khusus[cite: 9]

```
